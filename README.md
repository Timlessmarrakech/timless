# Morocco Travel Website (HTML)

This project contains a very small static website that loads placeholder travel data from JSON files. All pages are plain HTML and use a little vanilla JavaScript for randomizing content.

## Getting Started

Open `index.html` in your browser. No build step is required.

## Editing Data

The `data/en/` folder holds JSON files for tours, activities, accommodations, blog posts, testimonials and FAQs. Update or replace these files with your real data.

## Randomization

`script.js` shuffles the arrays before inserting cards on the page so the homepage feels a little different on each refresh.
